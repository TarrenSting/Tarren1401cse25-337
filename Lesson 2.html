CSS Basics

CSS (Cascading Style Sheets) is used to style and design web pages. It controls how HTML elements look on the screen.

What CSS Does

  • Changes colors
  • Controls layout
  • Adds spacing

Example

p {
    color: blue;
    font-size: 20px;
}

Ways to Use CSS

  • Inline CSS
  • Internal CSS
  • External CSS (best practice)

CSS makes websites visually appealing and improves user experience.
