HTML Basics

HTML (HyperText Markup Language) is the standard language used to create web pages. It defines the structure of a webpage using elements called tags.

Basic Structure of HTML

<!DOCTYPE html>
<html>
<head>
    <title>My Page</title>
</head>
<body>
    <h1>Hello World</h1>
</body>
</html>

Common HTML Tags

  • <h1> - <h6>: Headings
  • <p>: Paragraph
  • <a>: Links
  • <img>: Images

Example

<p>This is a paragraph</p>
<a href="https://example.com">Visit Site</a>

HTML is the foundation of all websites. It works together with CSS and JavaScript.
